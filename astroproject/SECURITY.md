# Security Policy — astroproject

AstroProject meta-package. This package contains no runtime code. Security issues should be reported to the relevant sub-package.

## Supported Packages

| Package | Status |
|---------|--------|
| astrocrawl | Active |
| astroframe | Alpha |
| astrobasis | Active |
| astroweave | Pre-release |

## Reporting

Report security vulnerabilities via GitHub Issues: https://github.com/Etoileint/AstroProject/issues
